# ✨ Todo List App - 现代化任务管理应用

一个简洁优雅的 Web 任务管理应用，集现代 UI、流畅动画和强大功能于一身。

![License](https://img.shields.io/badge/license-MIT-green)
![Version](https://img.shields.io/badge/version-1.0.0-blue)

## 🌟 核心特性

### 📋 任务管理
- ✅ **快速添加任务** - 支持回车键快速提交
- 📝 **编辑任务** - 随时修改任务内容
- 🗑️ **删除任务** - 带平滑动画的删除效果
- ✓ **完成标记** - 清晰的完成/未完成状态切换

### 🎯 高级功能
- **任务分类** - 工作、生活、学习、其他
- **优先级设置** - 高（🔴）/ 中（🟡）/ 低（🟢）三个优先级
- **标签系统** - 灵活的多标签分类
- **智能筛选** - 按状态、分类、优先级多维度筛选
- **任务统计** - 实时显示总数、待完成数、已完成数和完成率

### 🎨 界面设计
- **现代 UI** - 使用 Tailwind CSS 打造简洁视觉体验
- **毛玻璃效果** - Glassmorphism 设计风格
- **梯度背景** - 炫彩渐变背景
- **平滑动画** - 任务添加/删除时的淡入淡出效果
- **深浅主题** - 🌓 支持暗黑模式和亮色模式切换

### 💾 数据持久化
- **本地存储** - 使用 localStorage 自动保存数据
- **刷新无忧** - 页面刷新后数据完整恢复
- **主题记忆** - 记住用户的主题偏好

### 📱 响应式设计
- 完美适配桌面、平板和手机屏幕
- 触摸友好的交互设计

## 🚀 快速开始

### 安装
无需安装依赖！直接在浏览器中使用。

```bash
# 克隆仓库
git clone https://github.com/WYBupup/todo-list-app.git

# 进入项目目录
cd todo-list-app

# 用浏览器打开 index.html
open index.html
```

或者直接访问在线演示（如部署了的话）。

## 📖 使用指南

### 添加任务
1. 在输入框中输入任务内容
2. 选择任务分类（可选）
3. 设置优先级（可选）
4. 添加标签（可选）
5. 点击"➕ 添加任务"或按 Enter 键

### 管理任务
- **标记完成** - 点击复选框标记任务完成/未完成
- **编辑任务** - 点击 ✏️ 编辑按钮修改任务内容
- **删除任务** - 点击 🗑️ 删除按钮移除任务

### 筛选和查看
- **按状态筛选** - "全部" | "待完成" | "已完成"
- **按分类筛选** - 从分类下拉菜单选择
- **按优先级筛选** - 从优先级下拉菜单选择
- **清除筛选** - 点击"清除筛选"重置所有过滤条件

### 切换主题
点击右上角的 🌙/☀️ 图标在深浅主题间切换

## 💻 技术栈

| 技术 | 说明 |

## 📊 数据模型

任务对象结构：

```javascript
{
  id: Number,           // 唯一标识（时间戳）
  text: String,         // 任务内容
  completed: Boolean,   // 完成状态
  category: String,     // 分类
  priority: String,     // 优先级（low/medium/high）
  tags: Array,          // 标签数组
  createdAt: String     // ISO 格式时间戳
}
```
